# javalint

> A GitHub App built with [Probot](https://github.com/probot/probot) that analyse java source for each push

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```

## Usage
Go [here](https://github.com/apps/javalint) and enable the bot for your repositories.
